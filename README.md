# CareeroGuru

> Empowering Students. Shaping Careers.

CareeroGuru is a web and mobile platform that connects college students with verified mentors and counsellors through an appointment booking system.

## Vision

To simplify career guidance by making qualified mentors easily accessible to students.

## Tech Stack (Planned)

- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL
- Mobile: React Native
- Version Control: Git & GitHub
